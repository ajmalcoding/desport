# 🚀 DesPort – Designer Portfolio Platform

DesPort is a web application that allows designers to showcase their work and helps users discover designers based on skills, categories, and location.

---

## 📌 Project Overview

This project is built as a learning-focused full-stack application using Django.  
It demonstrates backend development, database design, and real-world feature implementation.

---

## 🛠 Tech Stack

- **Backend:** Python, Django  
- **Frontend:** HTML, TailwindCSS  
- **Database:** PostgreSQL  
- **UI Development:** Lovable AI (for faster UI design and implementation)
- **Tools:** VS Code, Git, GitHub  

---

## ✨ Features

- 👤 Designer portfolio system to showcase works and services  
- 🔍 Client-side search and filtering system to discover designers  
- 🎯 Filter designers by skills, categories, and location  
- 🖼 Project listing and detailed project view with multiple images  
- 🔗 Platform to connect designers and clients  

---

## 📸 UI & Frontend

The frontend UI is designed and implemented using **Lovable AI** to speed up development,  
with **TailwindCSS** used for styling and layout.
